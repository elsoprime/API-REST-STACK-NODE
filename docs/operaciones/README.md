## Regla de retomada

Antes de continuar trabajo nuevo:

- revisar `RETOMADA_CHECKLIST.md`
- confirmar si el cambio parte desde `main` o desde una rama `wip/*`
- abrir una rama `feat/*` nueva para desarrollo activo
